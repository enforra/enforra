# Changelog

## 0.1.0 — 2026-05-14

Initial open source release candidate.

- policy-core: policy loading, evaluation, grouped conditions, priority ordering, and decision trace
- policy-simulator: local policy test case runner
- sdk-node: `enforceToolCall` wrapper for agent tool callbacks
- local-audit: JSONL audit logging with redaction and optional hash-chain integrity
- examples: support refund, OpenAI-style wrapper, MCP-style policy, approval evidence, audit integrity, and benchmarks
- docs: threat model, SDK reference, policy testing, audit behavior, benchmarks, and architecture notes
